Café Robusta

Robusta Naturel

1-Récolte :
- Cueillir les cerises de café à maturité. 2-Tri :
- Trier les cerises pour enlever les fruits endommagés ou non mûrs. 3-Séchage :
- Étaler les cerises de café sur des lits de séchage (tables ou patios) en une seule couche.
- Remuer régulièrement pour assurer un séchage uniforme et éviter la fermentation non désirée.
- Sécher jusqu'à ce que les cerises atteignent une teneur en humidité d'environ 11-12%. 4-Décorticage :
- Passer les cerises séchées dans une décortiqueuse pour enlever la pulpe séchée et les parchemins. 5-Tri et Classification :
- Trier les grains décortiqués pour enlever les débris, les grains défectueux et les impuretés. 6-Stockage :
- Emballer les grains de café dans des sacs appropriés pour le stockage ou la vente.

Robusta Lavé

1-Récolte :
- Cueillir les cerises de café à maturité. 2-Tri :
- Trier les cerises pour enlever les fruits endommagés ou non mûrs. 3-Pulpage :
- Utiliser une machine à pulper pour enlever la pulpe des cerises. 4-Fermentation :
- Fermenter les grains dépulpés dans des cuves d'eau pendant 12 à 48 heures, selon les conditions climatiques.
- Vérifier régulièrement pour éviter une fermentation excessive. 5-Tri et Classification :
- Trier les grains décortiqués pour enlever les débris, les grains défectueux et les impuretés. 6-Stockage :
- Emballer les grains de café dans des sacs appropriés pour le stockage ou la vente.

Robusta Excelsa

1-Récolte :
- Cueillir les cerises de café à maturité. La variété Excelsa peut parfois nécessiter une attention particulière en raison de ses caractéristiques distinctes. 2-Tri :
- Trier les cerises pour enlever les fruits endommagés ou non mûrs. 3-Séchage :
- Sécher les grains comme décrit pour le traitement choisi (naturel ou lavé). 4-Décorticage :
- Décortiquer les grains pour enlever les couches externes. 5-Tri et Classification :
- Trier les grains décortiqués pour enlever les débris, les grains défectueux et les impuretés. 6-Stockage :
- Emballer les grains de café dans des sacs appropriés pour le stockage ou la vente.
