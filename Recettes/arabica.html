Arabica

Arabica Naturel

1-Récolte :
Les cerises de café mûres sont cueillies à la main. 2-Tri :
- Trier les cerises pour enlever les fruits endommagés ou non mûrs. 3-Séchage :
Les cerises entières sont étalées sur des lits de séchage au soleil. 4-Décorticage :
- Passer les cerises séchées dans une décortiqueuse pour enlever la pulpe séchée et les parchemins. 5-Tri et Classification :
Les grains sont triés pour éliminer les impuretés et les grains défectueux 6-Stockage :
- Emballer les grains de café dans des sacs appropriés pour le stockage ou la vente.

Arabica Honey

1-Récolte :
Les cerises de café mûres sont cueillies à la main. 2-Tri :
- Trier les cerises pour enlever les fruits endommagés ou non mûrs. 3-Séchage :
Les grains dépulpés avec leur mucilage sont étalés sur des lits de séchage. 4-Décorticage :
Une fois les grains séchés, ils sont décortiqués pour enlever le parchemin. 5-Tri et Classification :
Les grains sont triés pour éliminer les impuretés et les grains défectueux, puis emballés pour l'expédition. 6-Stockage :
- Emballer les grains de café dans des sacs appropriés pour le stockage ou la vente.

Arabica Monsooned

1-Récolte :
Les cerises de café mûres sont cueillies à la main et traitées par voie humide 2-Tri :
- Trier les cerises pour enlever les fruits endommagés ou non mûrs. 3-Séchage :
Les grains lavés sont séchés jusqu'à atteindre une teneur en humidité d'environ 11-12%. 4-Décorticage :
Une fois le processus de moussonnage terminé, les grains sont décortiqués pour enlever le parchemin. 5-Tri et Classification :
Les grains sont triés pour éliminer les impuretés et les grains défectueux, puis emballés pour l'expédition. 6-Stockage :
- Emballer les grains de café dans des sacs appropriés pour le stockage ou la vente.
